Summary: Latitude vs. X

This project was designed to analyze trends in weather data based on latitude. Data was gathered using the OpenWeatherMap API and dataset with of over 500 random locations across the world.

When the data was compiled, a Pandas dataframe was created and graphs were added to show Max Temperature, Humidity, Cloudiness, or Windspeed on the Y-Axis and Longitude on the X-Axis. The data for the graphs is provided in this website and an explanations of each graph.
